Grid Layout
1. Настроить основную сетку. Размеры блока .sidebar относятся к блоку .main как 1 к 4. Можно использовать как координаты элементов в грид-контейнере, так и грид-области. Отступы между грид-элементами должны быть по 20px.
2. Определить сетку внутри блока.main. Товары должны равномерно распределиться по трём столбцам. Расстояние между товарами должно быть 15px. Товаров может быть бесконечное количество. При создании сетки нужно учесть, что текст под товаром — динамический, то есть может иметь большой объем.
3. Добавить медиазапросы. На экранах менее 786px для блока с классом .page задать ширину блока calc(100% - 32px), чтобы слева и справа были отступы по 16px. Еще нужно, чтобы .sidebar больше не мешал просматривать каталог — перенести его под хедер, а сам список товаров растянется во всю ширину экрана. Чтобы карточки не были слишком маленькими, сделать 2 столбца вместо трёх.
